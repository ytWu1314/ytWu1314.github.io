208. 实现 Trie (前缀树)

思路

  1. Trie树插入操作，从root开始遍历trie树，如果不存在当前结点则创建一个结点，否则往下走，并且标记当前串
  2. 查询操作，从根节点开始往下遍历字符串，如果不存在直接返回false，否则返回最后一个标记点的bool值。

235. 二叉搜索树的最近公共祖先

思路:

  1. 把p一直放在左边，q放在右边
  2. 如果当前结点≥p,≤q 直接返回root
  3. 如果较大的数小于当前结点递归左子树
  4. 否则递归右子树

236. 二叉树的最近公共祖先

思路: 有一个state的二进数表示四种情况，11表示找到了p，q的最近公共祖先

  1. 如果当前结点为空，直接返回0
  2. 递归左子树，找到p |=01 (二进制)
  3. 找到q |= 10 (二进制)
  4. 递归右子树
  5. 如果state==3 && ans ==NULL , ans=root
  6. return state
